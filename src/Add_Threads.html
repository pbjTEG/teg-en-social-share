Add Threads to Share Links

Let the JavaScript use standard images or set your custom image in this <img> element.
Please ensure the id attribute is customThreads.

For example: <img src="https://upload.wikimedia.org/wikipedia/commons/7/7a/Threads_Logo.svg" id="customThreads" alt="Share on Threads" style="max-height: 100px">
Note: the style attribute above is just for convenience and won’t be copied to the generated link.
