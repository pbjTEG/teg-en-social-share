Add Mastodon to Share Links

Let the JavaScript use standard images matching the page settings or set your custom image in this <img> element.
Please ensure the id attribute is customMastodon.

For example: <img src="https://joinmastodon.org/logos/logo-purple.svg" id="customMastodon" alt="Free, open-source decentralized social media.">

The new link will allow the user to select (or set) their Mastodon instance. It will then share the “Share title” set in “Social share settings” and the page URL.

Select Your Mastodon Instance
Select “Other” to enter it yourself. You must be logged in to Mastodon.
